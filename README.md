Student Grade Management System
A simple Python project for managing students and their grades.
:rocket: Features
Add new students (name + grade)
Show all registered students
Calculate average grade
Check if a student passed or failed
Interactive menu
Simple and beginner-friendly python project
:books: Project Description (EN)
This Python program allows you to manage a list of students and their grades.
You can add students, view all students, calculate the average grade, and check each student’s status (pass/fail).
It is designed for beginners who want to practice:
Lists
Loops
Conditionals
User input
Basic algorithm design
